Setting Up RAXKIO: A Quick Tutorial

Welcome to RAXKIO, your ultimate tool for organizing and managing various tasks efficiently. Follow these simple steps to set up RAXKIO on your system.

Step 1: Install Requirements

Start by installing the necessary requirements. To do this:

Open the install_requirements.bat file located in the RAXKIO directory.
Simply double-click on the file to execute it.
This script will automatically install all the required dependencies for RAXKIO to function smoothly.

Step 2: Launch RAXKIO

Once you've installed the requirements, you're ready to launch RAXKIO. Here's how:

Navigate to the RAXKIO directory.
Look for the main executable file named RAXKIO.exe.
Double-click on RAXKIO.exe to launch the application.
